About

Mu2e DAQ Run Log Viewer

A web-based interface for querying and managing the Mu2e DAQ run-log database. Supports searching runs by time period, inspecting subrun statistics, looking up Event Window Tags, and manually entering run data.

Details

Project Mu2e Experiment — DAQ Group
Repository github.com/{{ github_repo }}
Framework Django 4.2 / Python 3.9+
Database PostgreSQL (production) · SQLite (development)

Available queries

  • Runs by time period — search for runs within a UTC date/time range
  • Run detail — view subruns, event counts, and timing for a specific run
  • EWT lookup — find the subrun(s) containing a given Event Window Tag or range
